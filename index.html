Sistema de Informação em Saúde Bucal (SISB)

SISB é um sistema interoperável com base nos padrões de troca de informação em saúde adotados pelo Brasil e completamente livre.

  • Desenvolvimento: Github

HealthDB

HealthDB (Health Database) é um sistema de gerenciamento de banco de dados cuja linguagem de definição de dados é a ADL e cuja linguagem de manipulação de dados é a AQL. O HealthDB tem como objetivo facilitar o desenvolvimento de Sistemas de Informação em Saúde (SISs) interoperáveis.

  • Informações: Home
  • Desenvolvimento: Github

Sistema de Apoio à Elaboração de Pareceres

Sistema de Apoio à Elaboração de Parecer (SAEP) tem como propósito auxiliar a edição de pareceres de processos de progressão, promoção e avaliação de estágio probatório da Universidade Federal de Goiás (UFG). O SAEP é desenvolvido no contexto da Fábrica de Software do Instituto de Informática (UFG).

  • Desenvolvimento: Home
